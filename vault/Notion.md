# Notion

Категория: Программа
Подробнее: Notion — универсальное рабочее пространство, объединяющее заметки, базы данных, задачи и вики в одном приложении. Работает как конструктор: каждая страница состоит из блоков, которые можно свободно комбинировать — текст, таблицы, канбан-доски, галереи, календари. Особенно мощен для построения личных систем и командных воркспейсов без кода.
Официальная страница: https://notion.so
Бесплатная версия: Бесплатно для индивидуального использования — неограниченные страницы, базовые базы данных, Notion Calendar. Ограничение: максимальный размер одного файла 5 МБ. AI-функции недоступны (только пробный период).
Теги: продуктивность, заметки, базы данных, задачи, совместная работа, AI
