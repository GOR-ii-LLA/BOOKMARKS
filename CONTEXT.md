## Скиллы

### book-content — агент базы знаний
- **Расположение:** `C:\Users\Work\.claude\skills\book-content\`
- **Команды:** исследуй / добавь / измени / исследуй заново
- **Форматы ввода:** `исследуй X`, `клод - X`, `программа - X`
- **Два режима сохранения:**
  - В проекте с computer use → автопуш через GitHub API
  - В обычном чате → выдаёт .md файл → watcher подхватит из vault/inbox/
- **Категории:** пока создана только Claude, остальные создаются в диалоге
- **Правила для Claude (5 предложений):**
  1. Суть, простым языком
  2. Плюсы с примерами из жизни
  3. Плюсы с примерами из жизни
  4. Сценарий интеграции с Claude
  5. Сценарий интеграции с Claude (или минус, если есть)
- **После "Подробнее":** Бесплатная версия → Лайфхаки (совместимость с другими записями в Claude) → Теги
- **Минус только если:** заброшен или есть намного лучший аналог
- Простой язык, плюсы с примерами из жизни

### book-design — дизайн и код
- **Расположение:** `C:\Users\Work\.claude\skills\book-design\`
- **Как работает:** читает файлы из репо → показывает план изменений → ждёт подтверждения → пушит через GitHub API → Actions деплоит
- **Область:** CSS, React-компоненты, стили, вёрстка, тема, навигация

### GitHub API
- Токен встроен в SKILL.md скиллов
- Права: Contents Read and write на репозиторий BOOK
- Срок действия: 90 дней (обновить при истечении)

### Watcher
- `watcher.py` работает в фоне через `BOOK-watcher.vbs` в автозагрузке Windows
- Следит за `vault/inbox/` каждые 10 секунд
- Парсит .md → обновляет entries.js → git push
- Используется когда скилл работает в обычном чате (без computer use)
