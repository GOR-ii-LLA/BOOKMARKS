## Что такое BOOK

Пользователь добавляет записи через Cowork, они автоматически появляются на сайте.

## Разделение зон ответственности

**Cowork** — только контент:
- Исследование и добавление записей (4 режима: исследуй / добавь вручную / измени / исследуй заново)
- Создание новых категорий в диалоге с пользователем
- Обновление шаблонов в SKILL.md
- Работа только с vault/ и plugin/skills/SKILL.md

**Этот чат (Claude)** — только код и дизайн:
- UI, стили, компоненты React
- Логика watcher.py и deploy.sh
- Структура приложения
- Новые функции

## Дизайн-решения

### Цвета
- Акцент: `#1D9E75` (зелёный)
- Claude-вкладка: `#D97757` (оранжевый, фирменный цвет Claude)
- Фон светлый: `#f7f6f3`
- Фон тёмный: `#141412`

### Шрифты
- Заголовок BOOK: `Unbounded` (Google Fonts)
- Основной: `Inter` (Google Fonts)

### Навигация
- Вкладки категорий сверху — отсортированы по алфавиту
- Первая вкладка всегда **Claude** (оранжевый цвет, иконка-астериск SVG)
- Вкладки появляются только если есть записи в категории
- Под вкладками — строка тегов (все реальные теги из записей)
- Теги: сначала русские по алфавиту, потом латиница

### Записи
- Список: иконка · название · теги · ссылка на сайт
- Клик — раскрывающаяся панель с описанием
- Клик на тег — страница тега со всеми записями

### Тема
- Переключатель светлой/тёмной темы в шапке справа

### Мобильная версия
- Оптимизировано для вертикального экрана телефона
- `font-size: 16px` на input (iOS не зумит)
- `-webkit-tap-highlight-color: transparent`

## entries.js — формат данных

```js
{
  id: 1,
  name: 'Название',
  category: 'программы',     // программы | лекарства | бады | товары | идеи | claude
  icon: '💻',                // 💻 | 💊 | 🌿 | 📦 | 💡
  link: 'site.com',          // без https://
  tags: ['#тег1', '#тег2'],  // с # в начале
  desc: 'Описание.',
  extra: 'Доп. информация.',
  note: 'Примечание.',
}
```

---

## watcher.py — как парсит файлы

Читает поля из .md файла:
- `Категория:` → category + icon
- `Официальная страница:` → link (убирает https://)
- `Подробнее:` → desc
- `Бесплатная версия:` + `Платная версия:` → extra
- `Альтернативы:` → note
- `Теги:` → tags (добавляет # к каждому)

Файл должен называться: `Название.md` (без даты)

## Автоматизация

### watcher.py
- Запуск: `python watcher.py` в папке BOOK/
- Следит за `vault/inbox/` каждые 10 секунд
- Парсит .md → обновляет entries.js → git push
- GitHub Actions деплоит сайт автоматически

### deploy.sh
- Запуск: `bash deploy.sh` в папке BOOK/
- Используется для деплоя изменений кода
- git add -A → commit → push
