## Скиллы

### book-content — агент базы знаний
- **Расположение:** `C:\Users\Work\.claude\skills\book-content\`
- **Команды:** исследуй / добавь / измени / исследуй заново
- **Как работает:** исследует объект → показывает отчёт → ждёт подтверждения → пушит .md + entries.js через GitHub API → сайт обновляется автоматически через Actions
- **Категории:** Claude, Программы, Лекарства, БАДы, Товары, Идеи
- **Особенности:**
  - Всегда спрашивает категорию перед исследованием
  - Для категории Claude: 2 предложения о плюсах + 3 о интеграции с Claude (или 2, если есть серьёзный минус)
  - Минус пишется только если: продукт устарел, есть лучший аналог, маленький бесплатный план, много негативных отзывов
  - Простой язык, плюсы с примерами из жизни

### book-design — дизайн и код
- **Расположение:** `C:\Users\Work\.claude\skills\book-design\`
- **Как работает:** читает файлы из репо → показывает план изменений → ждёт подтверждения → пушит через GitHub API → Actions деплоит
- **Область:** CSS, React-компоненты, стили, вёрстка, тема, навигация

### GitHub API
- Оба скилла используют GitHub Contents API с Personal Access Token (Fine-grained)
- Токен: `github_pat_11B6R4UPY...` (хранится в scripts/github_api.py внутри скиллов)
- Права: Contents Read and write на репозиторий BOOK
- Срок действия: 90 дней (обновить при истечении)

## Дизайн-решения

### Цвета
- Акцент: `#1D9E75` (зелёный)
- Claude-вкладка: `#D97757` (оранжевый, фирменный цвет Claude)
- Фон светлый: `#f7f6f3`
- Фон тёмный: `#141412`

### Шрифты
- Заголовок BOOK: `Unbounded` (Google Fonts)
- Основной: `Inter` (Google Fonts)

### Навигация
- Вкладки категорий сверху — отсортированы по алфавиту
- Первая вкладка всегда **Claude** (оранжевая кнопка, иконка-астериск SVG, белеет при активации)
- Вкладки появляются только если есть записи в категории
- Под вкладками — строка тегов (все реальные теги из записей)
- Теги: сначала русские по алфавиту, потом латиница — везде (в панели тегов и внутри записей)

### Записи
- Список: иконка · название · теги · ссылка на сайт
- Клик — раскрывающаяся панель с описанием
- Клик на тег — страница тега со всеми записями

### Тема
- Переключатель светлой/тёмной темы в шапке справа

### Мобильная версия
- Оптимизировано для вертикального экрана телефона
- `font-size: 16px` на input (iOS не зумит)
- `-webkit-tap-highlight-color: transparent`

## entries.js — формат данных

```js
{
  id: 1,
  name: 'Название',
  category: 'программы',     // программы | лекарства | бады | товары | идеи | claude
  icon: '💻',                // 💻 | 💊 | 🌿 | 📦 | 💡 | ✦
  link: 'site.com',          // без https://
  tags: ['#тег1', '#тег2'],  // с # в начале
  desc: 'Описание.',
  extra: 'Доп. информация.',
  note: 'Примечание.',
}
```

## Как вносить изменения

### Контент (записи)
Через скилл book-content в любом чате Claude:
1. Написать "исследуй [название]"
2. Выбрать категорию
3. Подтвердить отчёт
4. Запись автоматически появится на сайте

### Код и дизайн
Через скилл book-design в любом чате Claude:
1. Описать что изменить
2. Подтвердить план
3. Изменения автоматически появятся на сайте

### Ручной деплой (если нужно)
1. Внести изменения локально
2. `bash deploy.sh` из корня BOOK/
